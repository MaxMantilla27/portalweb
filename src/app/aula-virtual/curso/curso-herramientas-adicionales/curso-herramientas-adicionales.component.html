Herramientas adicionales
Denominación Link de descarga Descripción de uso
Apache Hadoop: Hbase https://hbase.apache.org/downloads.html Una base de datos distribuida no relacional diseñada para manejar grandes cantidades de datos estructurados. Es utilizada principalmente para aplicaciones que requieren alta disponibilidad y escalabilidad.
Arduino https://www.arduino.cc/en/software Una plataforma de prototipos electrónicos de código abierto basada en hardware y software fáciles de usar. Permite a los usuarios crear proyectos interactivos con microcontroladores.
Visual Studio Code (C#, .net, VB.net) https://code.visualstudio.com/download Un editor de código fuente ligero y potente, ideal para el desarrollo en múltiples lenguajes de programación. Soporta extensiones que amplían su funcionalidad, convirtiéndolo en un entorno de desarrollo flexible.
Java (NetBeans y Eclipse) https://www.java.com/es/download/ie_manual.jsp Una plataforma de desarrollo que permite crear aplicaciones en Java. Incluye IDEs populares como NetBeans y Eclipse que ofrecen herramientas robustas para la programación, depuración y gestión de proyectos Java.
Apache JMeter https://jmeter.apache.org/download_jmeter.cgi Una herramienta de prueba de carga y rendimiento que permite analizar y medir el rendimiento de aplicaciones y servicios web. Es extensible y soporta una variedad de protocolos para pruebas.
StarUML https://staruml.io/download/ Una herramienta de modelado UML utilizada para diseñar y analizar sistemas orientados a objetos. Facilita la creación de diagramas UML que son esenciales para la planificación y documentación de software.
Zephyr https://support.smartbear.com/zephyr-enterprise/downloads/ Una herramienta de gestión de pruebas que proporciona trazabilidad y visibilidad completas del ciclo de vida de las pruebas. Ofrece integración con otras herramientas de desarrollo y gestión de proyectos.
Unity3D https://store.unity.com/es/download Un motor de desarrollo de videojuegos y aplicaciones interactivas 3D en tiempo real. Permite a los desarrolladores crear experiencias inmersivas en diversas plataformas, incluyendo móviles, consolas y VR.
Microsoft SQL Server Express https://www.microsoft.com/es-es/download/details.aspx?id=101064 Un sistema de gestión de bases de datos relacional gratuito, ideal para aplicaciones pequeñas y medianas. Ofrece características básicas de SQL Server, adecuado para desarrollo y producción en menor escala.

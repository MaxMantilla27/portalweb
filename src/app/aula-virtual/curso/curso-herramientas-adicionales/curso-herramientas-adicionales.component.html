Herramientas adicionales
Este apartado sirve para que el alumno pueda acceder a las herramientas digitales que le permitirán el desarrollo de su Programa de estudios.
Esta sección ofrece una lista de herramientas adicionales que se han considerado para el desarrollo. Cada herramienta incluye un enlace de descarga y una breve descripción de su uso, proporcionando una referencia rápida para identificar y acceder a las herramientas más adecuadas para diversas necesidades de desarrollo y gestión de proyectos. Las licencias de esta relación de programas no requieren pago, puesto que son de uso libre. Podrás descargarlas e instalarlas sin mayores restricciones debes tener en cuenta los requisitos solicitados para cada una.

Denominación Link de descarga Descripción de uso Requisitos mínimos y recomendados
Apache Hadoop: Hbase https://hbase.apache.org/downloads.html Una base de datos distribuida no relacional diseñada para manejar grandes cantidades de datos estructurados. Es utilizada principalmente para aplicaciones que requieren alta disponibilidad y escalabilidad. Mínimos: RAM: 8 GB, Disco: 10 GB, Procesador: Multinúcleo (2 núcleos), Windows 10 o superior.
Recomendados: RAM: 16 GB, Disco: 50 GB, Procesador: Multinúcleo (4 núcleos), Windows Server 2016 o superior.
Arduino https://www.arduino.cc/en/software Una plataforma de prototipos electrónicos de código abierto basada en hardware y software fáciles de usar. Permite a los usuarios crear proyectos interactivos con microcontroladores. Mínimos: RAM: 2 GB, Disco: 1 GB, Procesador: 1 GHz, Windows 7 o superior.
Recomendados: RAM: 4 GB, Disco: 2 GB, Procesador: 2 GHz, Windows 10 o superior.
Visual Studio Code (C#, .net, VB.net) https://code.visualstudio.com/download Un editor de código fuente ligero y potente, ideal para el desarrollo en múltiples lenguajes de programación. Soporta extensiones que amplían su funcionalidad, convirtiéndolo en un entorno de desarrollo flexible. Mínimos: RAM: 2 GB, Disco: 200 MB,Procesador: 1.6 GHz, Windows 7 o superior.
Recomendados: RAM: 4 GB, Disco: 500 MB, Procesador: Multinúcleo, Windows 10 o superior.
Java (NetBeans y Eclipse) https://www.java.com/es/download/ie_manual.jsp Una plataforma de desarrollo que permite crear aplicaciones en Java. Incluye IDEs populares como NetBeans y Eclipse que ofrecen herramientas robustas para la programación, depuración y gestión de proyectos Java. Mínimos: RAM: 2 GB, Disco: 2 GB, Procesador: 1.6 GHz, Windows 7 o superior.
Recomendados: RAM: 4 GB, Disco: 4 GB, Procesador: Multinúcleo, Windows 10 o superior.
Apache JMeter https://jmeter.apache.org/download_jmeter.cgi Una herramienta de prueba de carga y rendimiento que permite analizar y medir el rendimiento de aplicaciones y servicios web. Es extensible y soporta una variedad de protocolos para pruebas. Mínimos: RAM: 2 GB, Disco: 1 GB, Procesador: 1 GHz, Windows 7 o superior.
Recomendados: RAM: 4 GB, Disco: 2 GB, Procesador: Multinúcleo, Windows 10 o superior.
StarUML https://staruml.io/download/ Una herramienta de modelado UML utilizada para diseñar y analizar sistemas orientados a objetos. Facilita la creación de diagramas UML que son esenciales para la planificación y documentación de software. Mínimos: RAM: 2 GB, Disco: 300 MB, Procesador: 1 GHz, Windows 7 o superior.
Recomendados: RAM: 4 GB, Disco: 1 GB, Procesador: Multinúcleo, Windows 10 o superior.
Zephyr https://support.smartbear.com/zephyr-enterprise/downloads/ Una herramienta de gestión de pruebas que proporciona trazabilidad y visibilidad completas del ciclo de vida de las pruebas. Ofrece integración con otras herramientas de desarrollo y gestión de proyectos. Mínimos: RAM: 4 GB, Disco: 1 GB, Procesador: 1.6 GHz, Windows 7 o superior.
Recomendados: RAM: 8 GB, Disco: 2 GB, Procesador: Multinúcleo, Windows 10 o superior.
Unity3D https://store.unity.com/es/download Un motor de desarrollo de videojuegos y aplicaciones interactivas 3D en tiempo real. Permite a los desarrolladores crear experiencias inmersivas en diversas plataformas, incluyendo móviles, consolas y VR. Mínimos: RAM: 8 GB, Disco: 15 GB, Procesador: Multinúcleo (4 núcleos), Windows 7 (SP1+) o superior.
Recomendados: RAM: 16 GB, Disco: 30 GB, Procesador: Multinúcleo (8 núcleos), Windows 10 (64-bit) o superior.
Microsoft SQL Server Express https://www.microsoft.com/es-es/download/details.aspx?id=101064 Un sistema de gestión de bases de datos relacional gratuito, ideal para aplicaciones pequeñas y medianas. Ofrece características básicas de SQL Server, adecuado para desarrollo y producción en menor escala. Mínimos: RAM: 4 GB, Disco: 6 GB, Procesador: 1.4 GHz, Windows 10 o superior.
Recomendados: RAM: 8 GB, Disco: 10 GB, Procesador: Multinúcleo, Windows Server 2016 o superior.
